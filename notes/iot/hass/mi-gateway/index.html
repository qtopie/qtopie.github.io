小米网关刷机

April 7, 2026

设备说明

  • 型号：小米多模态网关第一代（MGL03）
  • 支持：Wi-Fi / ZigBee / 蓝牙 Mesh
  • 本文记录的是 UART 刷机 路线，不是纯软件开启 Telnet 的方式

拆机和接线提醒

  • 外壳拆解可参考这个视频：YouTube 拆机视频
  • 拆 PCB 时要注意排线，不要直接暴力撬
  • UART 转接板必须工作在 3.3V，不要给主板额外喂 VCC
  • 如果不想折腾焊接，建议直接买弹簧夹 / pogo clip，能省掉很多接触不良排查时间

我第一次拆的时候比较急，拆坏过一根 Wi-Fi 天线。机器虽然还能用，但不建议复刻这种操作。

assets 目录怎么用

./assets 里已经放好了这次刷机最关键的文件，不需要再到处翻链接。可以直接按自己的系统挑文件：

文件用途是否关键
assets/README.md原始刷机说明，包含接线图、Windows / Linux 操作方式必看
assets/migateway-v1.png设备示意图，正文配图使用参考
assets/rtkboot_115200.bin中间引导程序，刷机脚本会先把它打进设备必需
assets/mgl03_1.4.7_0160_mod20210309.uart可刷入的 UART 固件包，我这里先用它把设备拉到可用状态必需
assets/mgl03_uart_recovery.expectLinux 下的自动刷机脚本Linux 必需
assets/mgl03_uart_recovery.ttlWindows 下给 Tera Term 用的宏脚本Windows 必需

最小文件组合

  • Linux 刷机：rtkboot_115200.bin + mgl03_*.uart + mgl03_uart_recovery.expect
  • Windows 刷机：rtkboot_115200.bin + mgl03_*.uart + mgl03_uart_recovery.ttl

使用建议

  • rtkboot_115200.bin 是速度和稳定性的折中版本，通常够用
  • mgl03_1.4.7_0160_mod20210309.uart 是我实际使用的起步固件；有些新版本资料里不再直接附带 .uart 包，所以我先刷这个老版本
  • 官方脚本我这里遇到过小问题，assets 里的文件已经整理成可直接配合刷机使用的版本

刷机流程

  1. 按 assets/README.md 的接线说明连好 UART。
  2. 准备好 rtkboot_115200.bin、一个 .uart 固件包，以及与你系统匹配的脚本文件。
  3. Linux 下执行 mgl03_uart_recovery.expect，Windows 下用 Tera Term 打开 mgl03_uart_recovery.ttl。
  4. 按脚本提示完成刷写。
  5. 启动完成后做一次恢复出厂设置（连续按设备按键 10 次）。

刷完之后

刷机完成后，默认可以直接通过下面的方式登录：

telnet <IP>

默认用户名是 admin，无需密码。

如果只是先刷入过渡版本，后续我再用这里的固件仓库继续升级到较新的版本：
