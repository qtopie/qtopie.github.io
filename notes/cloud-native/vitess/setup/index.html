Quick guide for vitess

addons

microk8s enable dns hostpath-storage ingress

pull images behind firewall

https_proxy=socks5://192.168.50.1:1080 microk8s.ctr image pull registry.k8s.io/ingress-nginx/controller:v1.5.1

check resources

# list crd
mkctl get crd

# list all resource names
mkctl api-resources --verbs=list -o name

# list viteness cluster kind resource
mkctl get VitessCluster
